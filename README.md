# Индексатор папок

Консольная программа на Java. Сканирует папку, сохраняет список файлов в SQLite и ищет дубликаты.

## Возможности

- индексация файлов и папок (путь, размер, SHA-256);
- поиск одинаковых файлов;
- история сканирований.

## Требования

- Java 17 или новее

## Запуск

```bash
./run.sh
```

Скрипт скачает драйвер SQLite, соберёт проект и откроет меню.

База данных создаётся в файле `index.db` в этой же папке.

## Структура проекта

```
src/
  Main.java      — меню и логика программы
  Database.java  — работа с SQLite
```

## Меню

1. Проиндексировать папку  
2. Найти дубликаты  
3. История сканирований  
0. Выход
